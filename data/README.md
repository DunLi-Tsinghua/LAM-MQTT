# Data Directory

Raw datasets are not redistributed in this repository.

Place downloaded raw files in:

```text
data/raw/MQTTEEB-D/
data/raw/MQTT-IoT-IDS2020/
data/raw/IoT-23/
```

Intermediate and processed files are generated locally and are ignored by Git:

```text
data/interim/
data/processed/
```

See `docs/DATASETS.md` for official sources and expected layouts.
