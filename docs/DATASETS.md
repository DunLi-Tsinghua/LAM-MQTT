# Datasets

Raw datasets are not redistributed in this repository. Download each dataset from its official source and place the files in the expected folder.

## MQTTEEB-D

- Dataset name: MQTTEEB-D: A Real-World IoT Cybersecurity Dataset for AI-Powered Threat Detection in MQTT Networks.
- Official source: Mendeley Data, https://data.mendeley.com/datasets/jfttfjn6tr/1
- How to download: download the dataset package from the Mendeley Data page according to its terms.
- Expected folder: `data/raw/MQTTEEB-D/`
- Automatic download supported: partially, only when a direct no-login source is available.
- Raw files redistributed here: No.

## MQTT-IoT-IDS2020

- Dataset name: MQTT-IoT-IDS2020: MQTT Internet of Things Intrusion Detection Dataset.
- Official source: https://pureportal.strath.ac.uk/en/datasets/mqtt-iot-ids2020-mqtt-internet-of-things-intrusion-detection-data/
- Access note: IEEE DataPort or related mirrors may require login or access approval.
- Primary feature file used in this study: `biflow_features.zip`.
- Optional supplementary file: `uniflow_features.zip`.
- Expected folder: `data/raw/MQTT-IoT-IDS2020/`
- Raw files redistributed here: No.

## IoT-23

- Dataset name: IoT-23.
- Official source: Stratosphere Laboratory, https://www.stratosphereips.org/datasets-iot23
- Data used: labelled Zeek flow logs.
- Runtime note: some large logs may be skipped depending on the configured size threshold.
- Expected folder: `data/raw/IoT-23/`
- Raw files redistributed here: No.
