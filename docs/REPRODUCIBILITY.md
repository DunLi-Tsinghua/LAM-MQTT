# Reproducibility

## 1. Clone Repository

```bash
git clone https://github.com/DunLi-Tsinghua/LAM-MQTT.git
cd LAM-MQTT
```

## 2. Create Environment

Using pip:

```bash
pip install -r requirements.txt
```

Using conda:

```bash
conda env create -f environment.yml
conda activate lam-mqtt
```

## 3. Download Datasets

Download raw datasets from official sources:

- MQTTEEB-D: https://data.mendeley.com/datasets/jfttfjn6tr/1
- MQTT-IoT-IDS2020: https://pureportal.strath.ac.uk/en/datasets/mqtt-iot-ids2020-mqtt-internet-of-things-intrusion-detection-data/
- IoT-23: https://www.stratosphereips.org/datasets-iot23

Raw datasets are not redistributed here.

## 4. Place Raw Files

```text
data/raw/MQTTEEB-D/
data/raw/MQTT-IoT-IDS2020/
data/raw/IoT-23/
```

For MQTT-IoT-IDS2020, `biflow_features.zip` is the primary file. `uniflow_features.zip` is optional.

## 5. Run Preprocessing

```bash
python code/inspect_datasets.py
python code/preprocess.py
python code/aggregate_flow_features.py
```

## 6. Build Feature Views

```bash
python code/build_feature_views.py
```

## 7. Run Leakage Audit

```bash
python code/audit_leakage.py
```

## 8. Run Within-Dataset Experiments

```bash
python code/train_baselines.py
```

## 9. Run Cross-Dataset and Paper-Level Experiments

```bash
python code/run_stage2B_experiments.py
```

The wrapper scripts `cross_dataset_eval.py`, `feature_family_ablation.py`, and `plot_results.py` call the same paper-level experiment driver because these outputs are produced together.

## 10. Generate Figures and Paper Tables

Figures and tables are generated by the paper-level experiment driver and stored under:

```text
results/metrics/
results/tables/
results/figures/
```

## Expected Outputs

- `results/metrics/within_dataset_main_results.csv`
- `results/metrics/mqtt_to_mqtt_cross_dataset_main_results.csv`
- `results/metrics/feature_family_ablation_results.csv`
- `results/metrics/multiseed_stability_revised_results.csv`
- `results/tables/feature_views_audited_metadata_main.csv`
- `results/tables/mqtt_cross_dataset_common_features_main.csv`
- `results/figures/*.pdf`
- `results/figures/*.svg`

## Runtime Notes

- Random seed: 42 unless otherwise specified.
- Large raw datasets are excluded from Git.
- Runtime depends on local CPU, disk speed, and dataset availability.
- IoT-23 processing may skip large logs depending on configured thresholds.

## Known Limitations

- MQTT-IoT-IDS2020 access may require login depending on the official source.
- Cross-dataset transfer is domain-sensitive and should not be interpreted as unconditional generalization.
- Metadata-only IDS does not decrypt TLS and does not replace authentication, access control, or protocol hardening.
